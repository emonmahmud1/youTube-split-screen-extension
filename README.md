# YouTube Split Screen Extension

A Chrome extension that enhances your YouTube viewing experience with split screen layout.

## Features

### 🎬 Split Screen Mode
- Video player stays fixed on the left (70% width)
- Related videos and comments scroll on the right (30% width)
- Smooth scrolling with fixed video position
- Responsive design for different screen sizes

## Installation

1. Clone or download this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked" and select the extension folder
5. The extension icon will appear in your toolbar

## Usage

1. **Access Controls**: Click the extension icon in your Chrome toolbar
2. **Split Screen**: Toggle "Enable Split Screen" to activate/deactivate split layout
3. **Scroll**: Scroll down to see the video stay fixed on the left
